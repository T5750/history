### 故宫博物院
故宫博物院，是一座特殊的博物馆。成立于1925年的故宫博物院，建立在明清两朝皇宫——紫禁城的基础上。历经六百年兴衰荣辱，帝王宫殿的大门终于向公众敞开。故宫博物院绝无仅有的独特藏品，是世界上规模最大、保存最完整的紫禁城木结构宫殿建筑群。
* [https://www.dpm.org.cn/](https://www.dpm.org.cn/)
* [图书馆](https://www.dpm.org.cn/librarys.html)
